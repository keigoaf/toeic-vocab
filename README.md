# TOEIC 単語トレーナー (PWA)

TOEICの単語・熟語を **間隔反復（Leitnerボックス方式）** で暗記するためのアプリです。
スマホのホーム画面に置いて、ネイティブアプリのように使えます（オフライン対応・端末内に進捗を永続保存）。

最終更新: 2026-07-23 18:15 (JST) / バージョン **1.9.0**

---

## 主な機能

- **間隔反復**: 正解した語は上のボックスへ進み、次の出題までの間隔が延びる。不正解ならBox1へ戻る。
- **出題順（v1.4.0）**: 期限が来た語のうち、ボックスの大きい順（Box7→Box1）に出題。上位が尽きたら下位へ進み、最後にBox1。同一ボックス内は期限が古い順。
- **出題周期の設定**: 各ボックスの「次回出題まで何日後か」を設定画面から自由に変更可能（初期値: 当日/1/3/7/14/30/60日）。
- **単語管理**: 追加・編集・削除・検索。単語には **品詞**（名詞・動詞など）を付けられ、めくった裏面に表示。
- **一括インポート / エクスポート**: テキストの貼り付けでまとめて登録・差し替え・バックアップ。
- **JSON一括取り込み**（v1.3.0）: 「単語」→「一括インポート」から、同梱の **TOEIC頻出500問**（`toeic-500.json`）をワンタップ追加。自作の JSON ファイル読み込みにも対応。重複（同じ単語）は自動でスキップ。
- **発音（v1.6.0）**: カードをめくった裏面のスピーカーボタン（🔊）で、単語と例文を読み上げます。ブラウザ標準の音声合成を使うため追加ファイル・通信は不要で、オフラインでも動作します。
- **例文の和訳（v1.7.0）**: 裏面の例文の下に日本語訳を表示。同梱1500問すべてに訳が付いています。
- **バックアップ / 復元（v1.7.0）**: 設定画面から、学習の進み具合（ボックス・出題日・正誤回数）ごとJSONに保存でき、復元もできます。端末変更やデータ消失に備えられます。
- **進捗の永続保存**: ブラウザの `localStorage` に保存。アプリを閉じても残る（同一URL内で保持）。

## スマホで「アプリ」として使う手順

ホーム画面に本物のアプリのように入れるには、**HTTPSで配信**する必要があります（Service Workerの制約）。
一番手軽なのは **GitHub Pages（無料・即HTTPS）** です。

### 1. GitHub Pages に配置する

**Git CLI の場合:**

```bash
cd toeic-vocab
git init
git add .
git commit -m "TOEIC vocab PWA v1.2.0"
git branch -M main
git remote add origin https://github.com/keigoaf/toeic-vocab.git   # ← 自分のリポジトリに
git push -u origin main
```

その後、GitHub のリポジトリで
`Settings → Pages → Build and deployment → Source: Deploy from a branch`
→ Branch を `main` / `/(root)` にして **Save**。

1分ほどで次のURLで公開されます（ユーザー名・リポジトリ名に合わせて読み替え）:

```
https://keigoaf.github.io/toeic-vocab/
```

### 2. ホーム画面に追加する

公開URLをスマホのブラウザで開き、ホーム画面に追加します。

- **iPhone (Safari)**: 共有ボタン → 「ホーム画面に追加」
- **Android (Chrome)**: メニュー(⋮) → 「アプリをインストール」または「ホーム画面に追加」

追加後はアイコンから全画面で起動し、オフラインでも動きます。

### （代替）自分のVPSで配信する場合

`168.144.41.110` 等の自前サーバでも配信できますが、Service Worker を有効にするには
**有効なHTTPS証明書**が必要です（自己署名不可）。Let's Encrypt などで証明書を用意し、
静的ファイルとして上記フォルダを配信してください。手軽さではGitHub Pagesが上です。

## 一括インポート / エクスポートのデータ形式

1行に1語。カンマ（または タブ）区切りで、**英語 , 意味 , 品詞/種別 , 例文** の順。
3列目・4列目は省略可能です。

- 3列目に **品詞**（名詞・動詞・形容詞・副詞・前置詞・接続詞・代名詞、または `n.` `v.` `adj.` 等）を書くと単語として登録。
- 3列目に **`熟語` または `idiom`** を書くと熟語として登録（品詞なし）。
- 空欄なら単語（品詞なし）として登録。

例:

```
acquire, 獲得する, 動詞, We plan to acquire a firm.
comply with, 順守する, 熟語
invoice, 請求書, 名詞
tentative, 仮の, adj.
```

エクスポートも同じ形式で出力されるので、そのままインポートに貼り戻せます（往復可能）。

### JSONファイルの形式（v1.3.0）

オブジェクトの配列です。`term` と `meaning` が必須、他は任意。熟語は `"type":"idiom"` を付けます。

```json
[
  { "term": "acquire", "meaning": "獲得する", "type": "word", "pos": "動詞", "example": "They acquired a rival firm." },
  { "term": "comply with", "meaning": "順守する", "type": "idiom", "example": "Please comply with the rules." }
]
```

取り込み方法は3通り（「単語」→「一括インポート」内）:
1. **TOEIC頻出500問を追加** / **追加1000問を追加** / **追加54問を追加** ボタン（同梱データをワンタップ。合計1554問）
2. **JSONを貼り付けて取り込む**（AI出力をそのまま貼り付け。`\`\`\`json` の囲みは自動除去）
3. **JSONファイルを読み込む**（`.json` を選択）

既存と同じ単語は自動でスキップされるので、重ねて取り込んでも増えません。

### AIに問題を作らせるプロンプト

無料のAIに以下を渡すと、そのまま貼り付け取り込みできるJSONが得られます（`【个数】`と`【テーマ】`は書き換え）。出力後、必要なら誤記チェックをしてから貼り付けてください。

```
あなたはTOEIC対策の英単語教材を作る専門家です。
以下の条件でTOEIC頻出の英単語・熟語を【50】個、JSON配列だけで出力してください。

# 出力形式（厳守）
- 出力はJSON配列のみ。前後に説明文やコードフェンス(```)を付けない。
- 各要素は次のキーを持つオブジェクト:
  - "term": 英語の見出し（単語または熟語）
  - "meaning": 日本語の意味（簡潔に。複数の意味は「/」で区切る）
  - "type": "word"（単語）または "idiom"（熟語・句動詞）
  - "pos": 品詞。typeがwordのときのみ。次のいずれか:
           "名詞" "動詞" "形容詞" "副詞" "前置詞" "接続詞" "代名詞"
           typeがidiomのときは "pos" を省略する。
  - "example": その語を使った短い英語例文（10語程度、TOEICらしいビジネス文脈）

# 内容の条件
- TOEIC(600〜800点帯)で頻出の語を優先。
- テーマ: 【ビジネス一般】
- 意味・品詞・例文に誤りがないこと。例文中で必ずtermを使う。
- 同じtermを重複させない。

# 出力例（この形式に厳密に従う）
[
  {"term":"acquire","meaning":"獲得する","type":"word","pos":"動詞","example":"They acquired a rival firm."},
  {"term":"comply with","meaning":"順守する","type":"idiom","example":"Please comply with the rules."}
]
```

## バックアップと注意点

- 進捗は `localStorage` に保存されます。これは **配信URL（ドメイン）ごと** に保持されます。
  配信先URLを変えると以前のデータは引き継がれません。
- 大事な単語リストは「単語」→「エクスポート」でテキストを控えておくと安全です。
  復元は「一括インポート」に貼り付けます。
- ブラウザの履歴・サイトデータを消すと `localStorage` も消えます。定期的なエクスポートを推奨。

## アプリの更新のしかた

**ホーム画面への再追加は不要です。** アイコンから開けば、オンライン時に自動で最新版へ切り替わります（v1.8.0以降）。

更新が反映されないときは、次の順に試してください。

1. アプリ内の **設定 → 「🔄 最新版に更新する」** を押す（学習履歴は消えません）
2. Safari で `https://<ユーザー名>.github.io/toeic-vocab/?v=2` のように末尾に `?v=数字` を付けて開く（数字は毎回変える）
3. それでも直らない場合のみ、設定 → Safari → 詳細 → Webサイトデータ から該当サイトを削除
   （※ 学習履歴も消えるため、事前に設定画面からバックアップを取ること）

開発側では、ファイルを更新したら `sw.js` の `CACHE` のバージョンを必ず上げてください。
